Условия использования

1. Общие положения

Данный сайт предназначен для публикации руководств и гайдов по игровому режиму CSC (Cristalix Survival Championship) на проекте Cristalix в игре Minecraft.

2. Назначение сайта

Сайт создан исключительно в образовательных целях для:

  • Публикации руководств по персонажам CSC режима
  • Обмена игровыми стратегиями и тактиками
  • Создания сообщества игроков CSC
  • Помощи новичкам в освоении режима

3. Права на контент

Все руководства и гайды создаются на основе:

  • Публично доступной информации об игровом режиме
  • Личного игрового опыта авторов
  • Анализа игровой механики CSC

Сайт не претендует на права интеллектуальной собственности проекта Cristalix.

4. Ответственность пользователей

Пользователи обязуются:

  • Не публиковать материалы, нарушающие правила Cristalix
  • Соблюдать этику общения в сообществе
  • Не распространять конфиденциальную информацию
  • Уважать авторские права

5. Отказ от ответственности

Администрация сайта не несет ответственности за:

  • Изменения в игровой механике CSC
  • Действия пользователей на проекте Cristalix
  • Актуальность опубликованных руководств

1. Сбор информации

Мы собираем только необходимую информацию:

  • Имя пользователя (никнейм)
  • Адрес электронной почты
  • Данные профиля (аватар, статистика форума)
  • Техническая информация (IP-адрес, браузер)

2. Использование данных

Собранная информация используется для:

  • Функционирования пользовательских аккаунтов
  • Модерации контента и форума
  • Улучшения работы сайта
  • Связи с пользователями по важным вопросам

3. Защита данных

Мы обеспечиваем защиту ваших данных:

  • Пароли хранятся в зашифрованном виде
  • Доступ к данным имеют только администраторы
  • Регулярное резервное копирование
  • Защита от несанкционированного доступа

4. Передача данных третьим лицам

Мы НЕ передаем ваши данные третьим лицам, за исключением:

  • Требований законодательства
  • Защиты прав и безопасности пользователей
  • Технических партнеров (хостинг-провайдеры)

5. Права пользователей

Вы имеете право:

  • Просматривать свои данные в профиле
  • Изменять личную информацию
  • Удалить свой аккаунт
  • Отозвать согласие на обработку данных

Данная политика не противоречит правилам проекта Cristalix. Сайт создан как независимый ресурс сообщества для образовательных целей.
