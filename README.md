# WinxTools Backend Setup

## Требования
- Node.js (версия 14 или выше)
- npm (устанавливается с Node.js)

## Установка Node.js
1. Скачайте Node.js с https://nodejs.org/
2. Установите, следуя инструкциям установщика
3. Проверьте установку: откройте командную строку и введите `node --version`

## Запуск проекта

### Способ 1: Автоматический (Windows)
Дважды кликните на файл `start-backend.bat`

### Способ 2: Ручной
1. Откройте командную строку (cmd) или PowerShell
2. Перейдите в папку проекта: `cd d:\gravebirth`
3. Перейдите в папку backend: `cd backend`
4. Установите зависимости: `npm install`
5. Запустите сервер: `npm start`

## Проверка работы
После запуска откройте браузер и перейдите на http://localhost:5500

## Остановка сервера
Нажмите Ctrl+C в командной строке где запущен сервер
